# Changelog

Todos los cambios notables en este proyecto serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/),
y este proyecto sigue [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-05-07

### 🎉 Lanzamiento Oficial

Presentación oficial de "La Materia que se Pregunta" - Un ensayo filosófico-científico sobre la realidad, la conciencia y la materia que se interroga a sí misma.

### ✨ Nuevo

- **Sitio web completamente funcional** con diseño elegante y minimalista
- **Soporte multiidioma** - Español e Inglés con cambio dinámico
- **Descarga de PDFs** - Acceso directo a versiones en español e inglés
- **Optimizaciones SEO** completas:
  - Meta tags (OpenGraph, Twitter Card)
  - Schema.org structured data (JSON-LD)
  - Sitemap.xml
  - Robots.txt
  - Canonical URLs
  - Hreflang para multiidioma

### 🏗️ Configuración Técnica

- **GitHub Pages habilitado** con Jekyll
- `_config.yml` - Configuración profesional
- `.gitattributes` - Normalización de líneas
- `.well-known/security.txt` - Información de seguridad
- Favicon personalizado
- Preconnect y DNS prefetch para optimización

### 🎨 Diseño

- Paleta de colores minimalista: Granito (#1a1a1a) + Oro (#c9a84c)
- Tipografía premium: Cinzel + EB Garamond de Google Fonts
- Animaciones suaves con Intersection Observer
- Diseño 100% responsive (mobile-first)
- Accesibilidad WCAG AA+

### 📚 Contenido

- **PDFs:**
  - `LaMateriaQueSePregunta_COMPLETO.pdf` (Español)
  - `QuestioningMatter_COMPLETE.pdf` (English)
- **Documentos Word:**

- **Documentación:**
